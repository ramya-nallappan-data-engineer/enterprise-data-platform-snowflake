# Snowflake Streams

## Purpose

Snowflake Streams provide Change Data Capture (CDC) functionality.

Instead of scanning entire source tables, Streams capture only row-level changes.

Benefits

- Incremental Processing
- Reduced Compute Cost
- Faster ELT Pipelines
- Simplified Change Detection

Streams Used

- CUSTOMER_STREAM
- BOOKING_STREAM
- PAYMENT_STREAM
- HOTEL_STREAM
- REVIEW_STREAM
